## Selected projects

A selection of academic and team-based projects across AI, full-stack development, computer vision and embedded systems.

### [CV_RPS](https://github.com/JorgeGonzalezzz/CV_RPS)

**Real-time Rock–Paper–Scissors game powered by computer vision.**

A two-player hand-gesture game using HSV colour segmentation, Kalman-based tracking and contour analysis to recognise rock, paper and scissors in real time. Each match is exported to a Flask dashboard for reviewing scores, rounds, detection frames and tracking masks.

`Python` · `Computer Vision` · `Flask` · `Jupyter Notebooks`

---

### [Financial News Alert System](https://github.com/JorgeGonzalezzz/Financial-Alert-Generator)

**End-to-end NLP and deep learning pipeline for financial news alerts.**

The system extracts named entities, predicts financial sentiment and generates alerts from news articles using rule-based logic or an LLM. It includes BiLSTM-based NER and sentiment models, a shared multi-task encoder, optional chart-image captioning and reproducible training workflows.

`Python` · `NLP` · `Deep Learning` · `BiLSTM` · `LLMs`

---

### [TerryNator ChatBot](https://github.com/JorgeGonzalezzz/TerryNator-ChatBot)

**Full-stack chatbot application powered by locally hosted LLMs.**

A Dockerised monorepo with a Next.js and React frontend, Django REST API, PostgreSQL database and JWT authentication. It supports user accounts, chat management, usage tracking and communication with local Ollama models.

`Django` · `Next.js` · `React` · `Ollama` · `PostgreSQL` · `Docker`

---

### [How To Train Your Perceptron](https://github.com/JorgeGonzalezzz/How-To-Train-Your-Perceptron)

**Interactive web project explaining how a perceptron learns AND and OR logic gates.**

A visual tool exploring decision boundaries and the effect of weights, bias and threshold values on binary classification, together with the supporting theory and original academic report.

`HTML` · `Machine Learning` · `Interactive Visualisation` · `Numerical Methods`

---

### [Terry](https://github.com/JorgeGonzalezzz/Terry)

**Raspberry Pi robotic car with manual and autonomous line-following modes.**

An embedded robotics project combining infrared line sensors, ultrasonic obstacle detection and PWM motor control. The vehicle supports Bluetooth remote control, autonomous black-line following and a Flask dashboard for real-time monitoring.

`Python` · `Raspberry Pi` · `Embedded Systems` · `Robotics` · `Flask`
